# tmp-user (Angular 20)
Ready-to-run Angular project with a small shop demo.

## Requirements
- Node.js **20+** (Angular 20 requires Node 20)
- npm

## Setup
```bash
npm install
npm start
```
Open http://localhost:4200/

## Build for deploy
```bash
npm run build
```
Artifacts will be in `dist/tmp-user/browser/`.
